# Thrayee Studio - Premium Interior Design Website

A world-class, conversion-focused website for Thrayee Studio, an interior design studio based in Hyderabad, India.

## Features

- 🎨 Premium, minimal design with elegant aesthetics
- 📱 Fully responsive and mobile-first
- ⚡ Optimized performance and SEO
- 🎯 Conversion-focused with multiple CTAs
- 🌍 Location-specific landing pages
- 📸 Portfolio with masonry layout
- 🔄 Before/After image comparisons
- 📋 Interactive design process timeline
- 🎭 Interior style finder quiz
- 💬 Testimonials from real clients
- 📞 Multiple contact methods (Phone, WhatsApp, Email)

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Language**: TypeScript
- **Image Optimization**: Next.js Image component
- **SEO**: Server-side rendering with Next.js

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
git clone https://github.com/thraye/thrayee-studio-website.git
cd thrayee-studio-website
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the website.

### Build for Production

```bash
npm run build
npm start
```
